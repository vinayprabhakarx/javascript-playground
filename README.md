# JavaScript Playground

A repository for learning and experimenting with JavaScript concepts and features.

## Contents

- **Basics**: Variables, data types, operators
- **Functions**: Function declarations, arrow functions, closures
- **Objects & Arrays**: Manipulation and iteration
- **Async**: Promises, async/await, callbacks
- **DOM**: Browser APIs and event handling
- **ES6+**: Modern JavaScript features

## Getting Started

1. Clone this repository
2. Open files in your code editor
3. Run scripts using Node.js or a browser console

```bash
node script.js
```

## Usage

Each file contains standalone examples and exercises.
